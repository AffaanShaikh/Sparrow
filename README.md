# Sparrow: My image generation repository
![Preview](outputs/Sparrow1.jpg)

- A gradio app for Caption-guided image generation and editing using vision-language and diffusion models, local ver. runs with standalone local models.
- Generates captions from input images using a local transformers vision-language model, uses the image and caption as paired input to guide a diffusers inpainting model.
- Supports background regeneration, object removal or by inverting the mask object generation. 
- Available on my [**Huggingface Space**](https://huggingface.co/spaces/Affaan/Autocaptioning-with-Inpainting-ImgGen), [*Preview*](bggenpdf.pdf).
